# 使用小程序插件

# 添加插件

  1. 前往微信服务平台open in new window查找需要的小程序插件

  1. 假设我们使用腾讯位置服务城市选择器,则点击,进行添加

  1. 点击添加后,选择需要这个插件的小程序

  1. 添加成功的界面

  1. 由于腾讯服务器问题,可能会添加失败

6.如果添加失败,请移步小程序管理后台open in new window中进行添加

选择设置 --> 第三方设置 --> 插件管理 --> 点击添加插件按钮--> 在搜索框中搜索你需要的插件 --> 点击添加

# 开通定位服务

由于使用的插件中有定位功能,所以需要在小程序管理后台open in new window,开启定位服务.

  1. 查找腾讯位置服务 开发 --> 开发工具 --> 开发能力 --> 腾讯位置服务

  1. 开通位置服务

  1. 微信账号扫码授权

  1. 在腾讯位置服务中注册账号

  1. 进入腾讯位置服务官方open in new window,点击右上角控制台

  1. 在控制台中选择 应用管理--> 我的应用--> 创建应用 输入应用名称(随意).应用类型(建议选其他)

  1. 在应用中,点击添加key,书写表单

  1. 生成 key. 此时已经成功开启定位服务

9 此时, 在小程序管理后台open in new window中 设置 --> 第三方设置--> 第三方平台授权管理中查看到腾讯位置服务

# 使用插件

添加插件和开通定位服务完成之后,即可在自己的小程序中使用. 进入指定插件的详情页面, 选择接入文档,按照文档要求使用即可

  1. 在你的小程序中引入插件
// app.json
{
  "plugins": {
    "citySelector": {
      "version": "1.0.1",
      "provider": "wx63ffb7b7894e99ae" //这是城市选择器appid
    }
  }
}
  1. 城市选择器插件需要小程序提供定位授权才能够正常使用定位功能：
// app.json
{
  "permission": {
    "scope.userLocation": {
      "desc": "你的位置信息将用于小程序定位"
    }
  }
}
  1. 在需要使用插件的页面中定义如下代码

    假设/pages/index/index 页面中点击按钮,使用插件

Page({
  handler() {
    const key = 'EXXBZ-BLIKU-PYBVK-B4OMY-AST6J-CWB6J' // 使用在腾讯位置服务申请的key
    const referer = 'test' // 调用插件的app的名称(随便写)
    const hotCitys = '北京,深圳,上海,武汉,西安' // 用户自定义的的热门城市(随便写)

    wx.navigateTo({
      url: `plugin://citySelector/index?key=${key}&referer=${referer}&hotCitys=${hotCitys}`,
    })
  },
})

  1. 选择任何一个城市之后,会返回到pages/index/index页面. 在pages/index/index中定义代码,可以拿到选择的城市信息
const citySelector = requirePlugin('citySelector')
Page({
  // 从城市选择器插件返回后，在页面的onShow生命周期函数中能够调用插件接口，获取cityInfo结果对象
  onShow() {
    const selectedCity = citySelector.getCity() // 选择城市后返回城市信息对象，若未选择返回null
    console.log(selectedCity)
    /*
    {
        fullname: "深圳市"
        id: "440300"
        location: {latitude: 22.54286, longitude: 114.05956}
        name: "深圳"
        pinyin: (2) ["shen", "zhen"]
    }
    */
  },
  onUnload() {
    // 页面卸载时清空插件数据，防止再次进入页面，getCity返回的是上次的结果
    citySelector.clearCity()
  },
})
