# 注册小程序账号

小程序注册地址:open in new window

# 有账号

有账号,则直接微信扫码登录. 扫码微信应是之前注册小程序的微信

# 无账号

无账号, 则点击立即注册

  1. 点击立即注册后选择小程序

  2. 填写表单, 注意: 建议使用 163 或 QQ 邮箱并阅读邮箱表单项提示文本)

  3. 前往邮箱激活

  4. 点击链接激活

  5. 信息登记 (请选择个人,选择其他选项需要对应的证明材料)

  6. 个人信息登记(注意: 个人最多注册 5 个小程序,个体商户最多拥有10个 3.企业,媒体,政府等部门最多拥有50个)

  7. 注册成功

  8. 查看 appid

    在左侧导航栏中找到开发--> 开发管理 --> 开发设置

    Appid 是小程序的唯一标识符.未来开发小程序时使用
