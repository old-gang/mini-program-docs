# 使用 UI 组件库(Vant Webapp)

UI 组件库,是指封装了许多小程序常用结构样式组件的库.开发者可以直接引入并使用.这么做的好处是方便开发者,快速搭建小程序项目.

我要使用的 UI 组件库叫做Vant Webapp. 是由有赞前端团队开源的小程序 UI 组件库.

那么一般使用 UI 组件库,都要使用npm进行下载,所以首先要学习的是如何在小程序中使用 npm 下载的包

# 小程序中使用 npm 包

从小程序基础库版本 2.2.1 或以上、及开发者工具 1.02.1808300 或以上开始，小程序支持使用 npm 安装第三方包。

  • 小程序使用 npm 包的三个限制

    • 依赖 node.js 的包不能在小程序中使用
    • 依赖浏览器内置对象的包不能在小程序中使用
    • 依赖 C++插件的包不能再小程序中使用

  • 使用 npm 包的三个步骤

    • 使用 npm 下载包
    • 构建 npm 包
    • 构建完成后,即可使用

小程序中使用 npm 包步骤open in new window

# 利用 npm 安装 Vant Webapp 的步骤

  1. 在项目根目录通过 npm 安装 Vant Webapp

    npm i @vant/weapp -S --production

  2. 将 app.json 中的 "style": "v2" 去除 小程序的新版基础组件强行加上了许多样式，难以覆盖，不关闭将造成部分组件样式混乱。

  4. 构建 npm

    打开微信开发者工具，点击 工具 -> 构建 npm ->在项目根目录会看到多了miniprogram_npm文件夹,表示构建成功 即可引入组件。

    注意: 最新版本的微信开发者工具,本地设置中没有使用 npm 模块,所以也不需要勾选了

    注意: 如果后续还要下载其他的 npm 包,则每次下载完之后,都要重新点击构建 npm. 如果构建失败,则删除已有的miniprogram_npm文件夹,然后再次点击构建 npm

# 使用 Vant Webapp

以 Icon 图标为例

  1. 在 app.json 或 index.json 中引入组件

    // app.json
"usingComponents": {
 "van-icon": "@vant/weapp/icon/index"
}

  2. 在 index.wxml 中使用

    <!-- index.wxml -->
<van-icon name="success" />

  4. 如需选择其他图标或进行其他配置,请查看 Vant Webapp 官方文档

# 定制主题颜色

Vant Webapp 中可以利用css变量来进行主题颜色的定制

以button组件为例, button 组件的主题背景颜色是绿色,如图:

假设我们的项目当前主体背景颜色是粉色,则按照下面的步骤进行配置

  1. 在 Vant Webapp 文档中找到定制主题->样式变量->点击配置文件

  2. 在配置文件中找到 button 对应的 less 变量名并复制(Vant Webapp 内部使用 less 定义变量)

  3. 在 app.wxss 中，粘贴 less 变量并改写成 CSS 变量，即可对全局生效

    /**app.wxss**/
page {
  /* 按钮的字体颜色 */
  --button-primary-color: white;
  /* 按钮的背景颜色 */
  --button-primary-background-color: pink;
  /* 按钮的边框颜色 */
  --button-primary-border-color: #ccc;
}

  4. 在任何一个页面中使用<van-button type="primary">主要按钮</van-button>,按钮的背景主题颜色都变成了粉色
